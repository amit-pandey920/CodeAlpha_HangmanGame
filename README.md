# Hangman Game

## Description

This is a simple text-based Hangman game developed using Python as part of the CodeAlpha Internship Program.

The player has to guess a randomly selected word one letter at a time. The player is allowed a maximum of 6 incorrect guesses.

## Features

* Randomly selects a word from a predefined list.
* Allows the player to guess one letter at a time.
* Displays correctly guessed letters.
* Limits incorrect guesses to 6.
* Prevents duplicate guesses.
* Validates user input.
* Displays win and lose messages.

## Technologies Used

* Python
* Random module
* While loops
* If-else statements
* Lists and Strings
* Console Input/Output

## How to Run

1. Install Python on your computer.
2. Download or clone this repository.
3. Open the terminal or command prompt in the project folder.
4. Run the following command:

```bash
python hangman.py
```

## Sample Gameplay

```text
Welcome to Hangman!

The word has 7 letters
Word: _ _ _ _ _ _ _
Guess a letter: a
Correct!

Word: _ a _ _ _ _ _
Guess a letter: z
Wrong!
```
